Addition Is Commutative, Syntax Is Hard

I’m working on a piece of writing about syntax being hard and am using +-comm below an example. It’s semantically intuitive, but its analytic proof is not.

+-identᵣ : ∀ (a : ℕ) → a + zero ≡ a
+-identᵣ zero = refl
+-identᵣ (succ a) = cong succ (+-identᵣ a)

+-succ : ∀ (a b : ℕ) → (succ a) + b ≡ a + (succ b)
+-succ zero b =  refl
+-succ (succ a) b = cong succ (+-succ a b)

+-comm : ∀ (a b : ℕ) → a + b ≡ b + a
+-comm zero b = sym (+-identᵣ b)
+-comm (succ a) b = trans (cong succ (+-comm a b)) (+-succ b a)
